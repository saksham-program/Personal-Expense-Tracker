# Personal Expense Tracker (Python)

## About the Project
This is a simple command-line based expense tracker built using Python.
I created this project after learning Python basics to practice how real programs are structured.

The program allows the user to:
- Add daily expenses
- View all saved expenses

This project helped me understand how lists, dictionaries, loops, and functions work together in a real use case.

---

## Features
- Add expense amount, category, and note
- View all expenses in a list format
- Simple menu-driven program
- Runs in terminal

---

## Concepts Used
- Python variables and data types
- Lists and dictionaries
- Functions
- While loop and conditional statements
- User input handling
